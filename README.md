Diamond Number Pattern (Python)

This repository contains a Python program that generates a diamond-shaped number pattern using user input.  
The project focuses on logical thinking and pattern creation with nested loops.

Overview

• Takes number of rows as input  
• Prints a symmetric diamond number pattern  
• Uses ascending and descending number sequences  

Concepts Used

• Nested loops  
• Pattern programming  
• Console output formatting  
• Logical problem solving  

File Included

diamond.py

Purpose

This project is useful for Python beginners to practice loop-based pattern problems and strengthen core programming concepts.
